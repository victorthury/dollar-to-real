# dollar-to-real

Go server and client to consume dollar to real API

# Usage:

It'll be necessary 2 terminal windows, one for `client.go`
and other for `server.go`. From root do:

```
# terminal for client.go
cd client
go mod tidy
go run client.go
```

```
# terminal for server.go
cd server
go mod tidy
go run server.go
```
